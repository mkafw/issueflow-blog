## 核心特性

- ✅ **输入/输出解耦**：Vim 本地写作，GitHub Actions 自动发布
- ✅ **零配置工作流**：关闭 Issue 即发布
- ✅ **版本控制**：所有内容 Git 追踪
- ✅ **多平台部署**：GitHub Pages / Vercel / Netlify

| 层级 | 技术 |

| 渲染 | Next.js 15 + Fumadocs |
| 部署 | Vercel / GitHub Pages |
